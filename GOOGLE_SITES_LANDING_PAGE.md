# TrendTok AI - Google Sites Landing Page Setup

## Overview

This guide will help you create a professional landing page for TrendTok AI on Google Sites at **trendtokai.com**.

## Step 1: Create Google Site

1. Go to https://sites.google.com
2. Click "Create" → "Blank Site"
3. Name: "TrendTok AI"
4. Click "Create"
5. You now have a site at `sites.google.com/site/trendtokaisomeid`

## Step 2: Set Up Custom Domain (Optional)

1. In Google Sites, click Settings (gear icon)
2. Go to "Manage custom domain"
3. Verify domain ownership at your domain registrar
4. Point domain to Google Sites
5. Site will be available at trendtokai.com

## Step 4: Design Styling

Use Google Sites built-in themes:
1. Go to Theme (paintbrush icon)
2. Select "Modern" or "Minimalist" theme
3. Customize:
   - Primary Color: Cyan (#006970)
   - Secondary Color: Violet (#712ae2)
   - Accent Color: Coral (#ba1340)
   - Font: Segoe UI or system default

## Step 5: Add Dashboard Embed

On the "Dashboard" page:

```html
<iframe
  src="https://trendtok-dashboard.vercel.app"
  width="100%"
  height="800px"
  style="border: none; border-radius: 8px;">
</iframe>
```

**Alternative**: Use "Embed" gadget:
1. Click "+ Embed"
2. Choose "Webpage"
3. Paste URL: `https://trendtok-dashboard.vercel.app`
4. Set dimensions: Full width, 800px height

## Branding Guidelines for Site

### Colors
- Primary (Cyan): #006970
- Secondary (Violet): #712ae2
- Accent (Coral): #ba1340
- Background: #ffffff (light), #0f172a (dark)
- Text: #1a202c (light), #f1f5f9 (dark)

### Typography
- Headlines: Bold, 28-48px
- Body: Regular, 14-16px
- Button text: Semi-bold, 14px

### Imagery
- Use screenshots from the dashboard
- Use icons from Lucide or similar
- Use gradient backgrounds (cyan to violet)
- Maintain consistent spacing and alignment
